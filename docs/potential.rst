==================================
Potential
==================================

EXPLAIN 

List available potentials:

[potential] options
====================

**potential**:

    default: There is no default.

    options:

        ``mpmd``: Lennard-Jones

        ``lj``: Lennard-Jones

        ``ltb``: EXPLAIN

        ``ann``: EXPLAIN

        ``bigdft``: EXPLAIN

        ``vasp``: EXPLAIN

        ``coulomb``: EXPLAIN

        ``siesta``: EXPLAIN

**potential_sec**:

    default: There is no default.

    options: All options from potential are also available here.
